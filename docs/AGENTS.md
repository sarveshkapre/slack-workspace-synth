# AGENTS

## Working agreements
- Keep changes small and shippable.
- Run `make check` before shipping.
- Update `docs/CHANGELOG.md` for user-facing changes.

## Commands
- Setup: `make setup`
- Dev server: `make dev`
- Tests: `make test`
- Lint: `make lint`
- Typecheck: `make typecheck`
- Build: `make build`
- Gate: `make check`

## Conventions
- Python 3.11
- Keep dependencies minimal.
- Prefer deterministic behavior with explicit seeds.
